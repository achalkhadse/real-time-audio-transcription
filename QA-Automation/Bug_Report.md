# Bug Report

## Bug ID
BUG_001

## Title
Speech transcription text not displayed

## Priority
High

## Status
Open

## Description
The application detects voice input and displays audio visualization, but spoken words are not converted into text on the screen.

## Steps to Reproduce
1. Open the application
2. Click Start
3. Allow microphone access
4. Speak clearly

## Expected Result
Spoken words should appear as text in real time.

## Actual Result
Only audio visualization is shown. No text output appears.

## Notes:
No error message is shown on the UI or browser console.

## Environment
- OS: Windows
- Browser: Chrome

## Severity
Major
